# ATM Transaction System (Python)

## 📌 Project Description

This project is a **simple ATM Transaction System built using Python**.
It simulates basic ATM operations such as checking balance, withdrawing money, and depositing money.
The program uses basic Python concepts like **functions, loops, and conditional statements**.

## 🚀 Features

* PIN authentication
* Check account balance
* Withdraw money
* Deposit money
* Exit the ATM system

## 🛠 Technologies Used

* Python
* Basic programming concepts (if-else, while loop, functions)

## ▶️ How to Run

1. Download or clone this repository.
2. Open the Python file in any Python environment (VS Code, Colab, or terminal).
3. Run the program:

```
python atm.py
```
